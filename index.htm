About

Welcome to BetaOnline, a Minecraft server that is aiming to strike a
balance between nostalgic and modern Minecraft. We offer a semi-vanilla
and old-school feel, with no big game-changing plugins, and no mods of any
kind. We have simple rules and you can do pretty much anything you want, as
long as it doesn't bother anyone else.

Update 29/03/22: To combat cheaters and griefers, a whitelist has been
added. Apply for membership of the server by filling out the form that is linked
on top of this page.

Update 16/04/22: To further combat the low quality of players, the server
has become invite-only. To be able to join the server, you need to know somebody on
the server to get an invite.

Update 25/08/22: The server is now public again. The building permission
system is back, so be sure to apply using our form.
